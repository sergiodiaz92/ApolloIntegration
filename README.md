# ApolloIntegration

Basic Integration of the Apollo API to get the contacts and upload them to a Postgres DB by using a console app.

## Solution

### Architecture and Design Implemented

- Clean Architecture

- Dependency Injection

- Mediator Pattern

- Option Pattern

### Frameworks, Package and Tools
  
    - .Net 6
    - EF Core
    - Fluent API
    - EF Migrations
    - Automapper
    - Npgsql
    - MediatR
    - FluentValidation
    - HttpClient
    - Newtonsoft.Json
    - Generic Host
